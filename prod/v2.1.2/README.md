## üöÄ Quick Start

### For Students
1. Open the application in any web browser
2. Enter your real first and last name (the app will guide you!)
3. Choose your adventure from the home screen
4. Start practicing multiplication!

### For Teachers (Classroom Mode)
1. Click "Battle mode" ‚Üí "Create classroom session"
2. Choose your game mode (Monster Race or Boss Battle)
3. Share the 4-character session code with students
4. Monitor progress on the live dashboard with proper student names
5. View final results and rankings

### For Students (Joining Classroom)
1. Click "Battle mode" ‚Üí "Join classroom session"
2. Enter the session code from your teacher
3. Wait for the teacher to start the battle
4. Compete with your classmates!
